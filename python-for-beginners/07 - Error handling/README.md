# Manipulação de erros

A manipulação de erros no Python é gerenciada através do uso de [try/except/finally](https://docs.python.org/3.7/reference/compound_stmts.html#except)

Python tem numerosos [built-in exceptions](https://docs.python.org/3.7/library/exceptions.html). 
Ao criar blocos `except`, eles precisam ser criados do mais específico ao mais genérico, de acordo com o [hierarchy](https://docs.python.org/3.7/library/exceptions.html#exception-hierarchy).
